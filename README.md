# WebdriverIO (WDIO) with Javascript

## Setup

> Skip to `Step 2` if using Gitpod

1. Initialize WDIO in the project

    ```bash
    # defaults
    npm init wdio . -- --yes
    ```

    ```bash
    # or go through the Setup Wizard manually
    npm init wdio
    ```

2. Run the example tests

    ```bash
    npm run wdio --spec example.e2e.js
    ```
